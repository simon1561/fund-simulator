# 股票模拟账户使用说明

这个文件给不懂代码的使用者看。日常只需要在飞书表格里录交易，然后双击文件夹里的重算入口。

## 第一次配置

1. 复制一份飞书 Base 模板，改成自己的名字，例如 `张三-股票模拟账户`。
2. 把这个项目文件夹放到电脑里，不要只拖出其中一个文件。
3. 双击 `setup.command`，按窗口里的提示完成安装和飞书授权。
4. 双击 `configure_account.command`。
5. 粘贴自己的飞书 Base 链接，按 Enter。
6. 看到“验证通过”后，配置就完成了。

如果第 3 步或第 5 步失败，把窗口截图发给 WorkBuddy 或项目维护者。

## 以后怎么用

1. 打开自己的飞书 Base。
2. 在 `投资标的库` 里新增自己要交易的股票。
3. 在 `交易流水` 里录入买入、卖出、分红、Put 等交易。
4. 回到这个项目文件夹，双击 `recalc_account.command`。
5. 窗口显示“重算完成”后，回到飞书看 `收益统计`、`持仓统计`、`资产快照`、`资产构成`。

## 需要记住的三件事

- 交易只在飞书 Base 里录。
- 录完交易后，双击 `recalc_account.command`。
- 报错时不要乱删文件，把报错截图发给 WorkBuddy 或项目维护者。

## 常见问题

### 提示没有找到项目虚拟环境

先双击 `setup.command`。

### 提示还没有配置 Base

先双击 `configure_account.command`，粘贴自己的 Base 链接。

### 提示没有权限或授权失败

重新双击 `setup.command`，按提示完成飞书授权。确认自己对这份 Base 有编辑权限。

### 提示缺少某张表

说明这份 Base 不是从模板复制出来的，或者改了表名。请重新复制模板，或者联系项目维护者。
