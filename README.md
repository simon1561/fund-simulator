# Fund Simulator / 股票模拟账户 MVP

「飞书多维表格 + 本地 Python 脚本」的个人股票模拟盘：在飞书表格里录交易，双击脚本实时取价、按 TWR 重算收益，再写回表格展示。**仅支持 macOS。**

> 每人用自己的飞书 Base（从同一模板各复制一份），互不影响。代码里不含任何账户数据。

## 一、首次安装

**不懂代码没关系**：把这一节连同仓库地址发给 WorkBuddy，让它照做即可。只需 macOS + 一个飞书账号（缺 Python / Node 时它会用 `brew install python node` 补上）。

1. 克隆仓库到一个**长期不动**的位置（别放「下载」或临时目录），推荐 `~/Documents/fund-simulator`：
   ```bash
   git clone https://github.com/simon1561/fund-simulator.git ~/Documents/fund-simulator
   ```
   不会用 git 就在 GitHub 页面 `Code → Download ZIP`，解压到同样位置。
2. 把维护者给你的**模板 Base** 复制到自己的飞书，改成自己的名字，确认有编辑权限。
3. 双击 `setup.command`——自动装好环境并引导飞书授权。
4. 双击 `configure_account.command`——粘贴你那份 Base 的链接，看到「验证通过」即完成。

图文版见 [`USER_GUIDE.md`](USER_GUIDE.md)；给 WorkBuddy 的提示词见 [`WORKBUDDY_PROMPT.md`](WORKBUDDY_PROMPT.md)。

## 二、日常使用

- **录交易**：在飞书 `交易流水` 表里追加（不改旧记录，录错就补一条 `冲正`）。新证券先在 `投资标的库` 建好。
- **刷新数据**：双击 `recalc_account.command`——用当天日期实时取价、重算持仓收益、写回表格。

## 三、交易怎么填

每种交易只填对应的几个格，其余留空：

| 交易类型 | 填这些 |
|---|---|
| 买入 / 卖出股票 | `数量`、`成交价`（原币） |
| 分红 | **一般不用手填**——现金分红重算时自动补录（见下）；要手录/覆盖才填 `数量` = 股数、`成交价` = 每股股息（原币） |
| 拆股 / 合股 | `拆合股比例`（1 拆 10 填 `10`，10 合 1 填 `0.1`） |
| 卖 Put（仅现金担保） | `合约数`、`行权价USD`、`到期日`、`权利金每股USD`（`合约乘数` 空 = 100） |
| 入金 / 出金 | `外部现金流USD`（入金正数，出金负数） |
| 冲正 | `现金影响USD`（直接填现金增减额） |

- **分红默认自动**：重算时脚本用 yfinance 抓取各持仓「买入日之后」的现金分红，自动补录进 `交易流水`（备注标「自动生成」）。你也可以手录，手录的**优先且不会被覆盖**（按证券+除息日去重）；想作废自动那笔，把它的 `数量` 改成 0（直接删会在下次重算被重建）。**实物分红**（派股票，如腾讯派京东）yfinance 没数据，需自己补。不想自动就加 `--no-auto-dividends`。
- `USD汇率`：留空则按交易日自动抓，手填以手填为准；美股填 1 或留空。
- `现金影响USD`：一般留空（脚本按 `数量 × 成交价 × 汇率` 自动算），只在想手动覆盖到账金额时填。
- `标的币种`：选好证券后自动带出，不用填。

## 四、表在算什么

- **持仓统计**：`买入均价原币` 是真实成交均价；`平均成本原币` 是**摊薄成本**（买入价扣掉累计分红），`盈亏额度 / 比例` 按摊薄口径算——分红直接体现在持仓盈亏里（和雪球一致）。`已实现盈亏USD` 只统计卖出损益。
- **资产构成 / 资产快照**：给收益统计看板用；各构成项金额加总 = 当前 NAV。
- **设置**（四个标签页）：`账户设置`（初始资金、TWR 起始日、回写的 NAV 等）、`基准设置`（恒生科技 / 沪深300 / 标普500 的代码和起始价）、`重算设置`（表格内触发按钮）、`使用说明`。

## 五、出问题 / 进阶

- 提示 **「行情抓取失败，沿用上次价格」**：多半是 Yahoo 临时限流，别短时间反复重算；用代理时让 `query1.finance.yahoo.com` 走稳定 / 直连节点。
- **重算指定某一天**：
  ```bash
  .venv/bin/python stock_sim_account_recalc.py --as-of 2026-06-24
  ```
- **想在飞书表格里点按钮重算**：先双击 `start_recalc_server.command`（保持终端窗口开着），再到 `设置 → 重算设置` 打开链接点按钮。
